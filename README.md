## Overview

This project implements a **Deterministic Finite Automaton (DFA)** that scans free-form text and identifies tokens matching the pattern:

```
[Number] [Unit]
```

## Steps to Run

1. Install dependencies:

```bash
pip install streamlit pandas
```

2. Run the app:

```bash
streamlit run app.py
```

The app will open at **http://localhost:8501** by default.
